## About
Mini program for generate a metro with different road and with mutliple parameter

## Features
- **Getter** of list of tunnel and station
- **Preview** of generation directly
  
## Parameter
- Deviation of tunnel
- Lenght of tunnel (and length min and max)
- Count of station
